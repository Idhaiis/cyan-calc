A modern cyan themed calculator built with Python & CustomTkinter.

Features
- Cyan themed UI
- Grid-based responsive layout
- Basic arithmetic operations

Tech Stack
- Python
- Tkinter
- CustomTkinter
- PyInstaller

Run Project
"bash
pip install tkinter
pip install customtkinter
python main.py
"

Build EXE
"bash
pyinstaller --onefile --windowed --icon=icon.ico main.py
"
